# 安装部署说明

## 1. 准备环境

- Python 3.10+
- Node.js 18+
- npm

## 2. 配置 Token

```bash
cd a-share-daily-review
cp backend/.env.example backend/.env
# 编辑 backend/.env，填写 TUSHARE_TOKEN
```

## 3. 确认股票池

股票池文件必须存在：

```text
data/universe/a_stock_universe.csv
```

## 4. 一键启动

Linux/macOS：

```bash
scripts/start.sh
```

Windows：双击或运行：

```bat
scripts\start.bat
```

## 5. 访问地址

- 前端：http://127.0.0.1:5173
- 后端健康检查：http://127.0.0.1:8000/api/health
