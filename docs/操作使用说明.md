# 操作使用说明

## 配置 Tushare Token

复制 `backend/.env.example` 为 `backend/.env`，填写 `TUSHARE_TOKEN`。

## 放置股票池

将 `a_stock_universe.csv` 放到：

```text
data/universe/a_stock_universe.csv
```

## 启动系统

```bash
scripts/start.sh
```

## 更新某天数据

打开前端，选择交易日，点击“更新数据”。如需忽略缓存，点击“强制刷新”。

## 查看复盘

选择已更新过的日期，点击“查询”。下方历史复盘可快速切换。

## 备份和恢复

备份：

```bash
scripts/backup.sh
```

恢复：

```bash
scripts/restore.sh backups/backup_xxx.tar.gz
```
